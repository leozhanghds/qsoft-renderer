# QSoftRenderer

#### 介绍
一个模仿OpenGL的软渲染器实现，初步支持了渲染管线，纹理贴图，MSAA反走样实现，支持glsl shader语言编程

![输入图片说明](image/run_result.png)

+ 相机
+ 顶点配置
+ 顶点着色器
+ 图元装配（三角形）
+ 光栅化
+ 片段着色器
+ 纹理贴图
+ MSAA反走样
+ 面剔除（基础）
+ 颜色混合（基础）
